# Investment Pitch Context: Physics-Inspired Probabilistic Portfolio Optimization

## **The Setup**

You're preparing for **investor presentations next week** to showcase revolutionary **probabilistic computing technology**. Investors are **not technically savvy** but need to understand the business importance and competitive advantage.

## **The Technology**

- **Hardware**: 4 real superconducting Niobium Nitride (NbN) nanowires called "Stochastic Binary Neurons" (SBNs)
- **Physics**: Exploits natural stochastic switching from superconducting transitions, vortex dynamics, and thermal fluctuations
- **Control**: Python software sets voltages via lock-in amplifier, reads probabilistic bit states back
- **Method**: QUBO (Quadratic Unconstrained Binary Optimization) encoding maps portfolio problems into physical energy landscapes

## **The Problem Being Solved**

Portfolio optimization is **NP-hard** - computational complexity explodes exponentially with number of assets. Classical computers struggle with large-scale optimization, often getting trapped in local optima.
