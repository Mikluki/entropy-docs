# Оптимизация инвестиционного портфеля с использованием физических стохастических бинарных нейронов (СБН): демонстрация аналоговой случайности

## Аннотация

В этой демонстрации показано, как стохастические бинарные нейроны (СБН) на основе ниобиевых нанопроводов решают прикладные задачи оптимизации. В качестве примера рассматривается оптимизация инвестиционного портфеля — выбор активов для максимизации доходности при контроле уровня риска. Демонстрация иллюстрирует несколько ключевых достижений:

1. **Подтверждение аналоговой случайности**: Разработанные нами устройства на основе ниобиевых нанопроводов функционируют как полноценные СБН с подлинной физически обусловленной случайностью.

2. **Решение практических задач**: СБН способны эффективно справляться со сложными задачами при использовании управления с обратной связью.

3. **Сравнительная эффективность**: Мы сопоставляем наш подход с классическими методами, такими как имитация отжига, и демонстрируем преимущества аппаратной случайности.

## Задача оптимизации

### Сложные ландшафты решений

Рассматриваемые задачи имеют сложный характер с множеством локальных минимумов. Цель состоит не в поиске идеального решения — что зачастую невозможно — а в быстром и надёжном нахождении хороших приближений.

Оптимизация портфеля относится к числу вычислительно сложных задач, таких как управление энергосетями, задача коммивояжёра и составление рабочих графиков. По мере увеличения размера задачи традиционные алгоритмы часто застревают в неэффективных решениях. Аналоговая случайность обеспечивает более глубокое и продуктивное исследование пространства решений.

### Основы оптимизации портфеля

Оптимизация инвестиционного портфеля требует выбора активов с учётом:

- Максимизации ожидаемой доходности
- Минимизации инвестиционных рисков
- Диверсификации по слабо коррелированным активам
- Соблюдения бюджетных ограничений

Это актуальная финансовая задача, вычислительная сложность которой резко возрастает с ростом числа доступных вариантов инвестирования.

## Преимущества стохастических бинарных нейронов

Предложенный метод использует физические стохастические бинарные нейроны (СБН), функционирующие под управлением механизма обратной связи, который:

1. Преобразует задачу в формат, совместимый с аппаратной реализацией
2. Динамически регулирует поведение каждого нейрона по мере продвижения оптимизации
3. Использует подлинную физическую (аналоговую) случайность для обхода сложных областей решений

Аналоговая случайность, обусловленная физическими флуктуациями в устройстве, может обеспечивать фундаментальные преимущества по сравнению с программной псевдослучайностью, особенно в задачах с множеством локальных минимумов, где качество случайных переходов существенно влияет на результат.

Эта демонстрация наглядно показывает, как физически реализованная случайность может быть эффективно использована для решения актуальных задач оптимизации в реальных приложениях.
